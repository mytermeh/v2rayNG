# v2rayNG Pro

A modified version of [v2rayNG](https://github.com/2dust/v2rayNG) with extra features and improvements.

[![GitHub Releases](https://img.shields.io/github/downloads/tempereddev/v2rayNG/latest/total?logo=github)](https://github.com/tempereddev/v2rayNG/releases)

## Download

Get the latest APK from the [Releases](https://github.com/tempereddev/v2rayNG/releases) page.

> Most phones need the **arm64-v8a** version.

## What's different

- Long-press on subscription tabs to rename them
- Fast server testing (worker-pool, no more freezing)
- Quick Test button on main screen
- Auto-update subscriptions (default: 15 min)
- Per-subscription update button
- Mux & Fragment Benchmark tool
- Routing mode toggle (Rule/Global/Direct) on main screen
- In-app update checker
- Auto-select best server after test

## Based on

This project is based on [2dust/v2rayNG](https://github.com/2dust/v2rayNG) and uses [Xray core](https://github.com/XTLS/Xray-core).
